# Variables

## Variables Types

- char (karakter) (1 byte)
  - işaretli -128 +127
  - işaretsiz 0 +255
- short (kısa tamsayı) (2 byte)
  - işaretli -32768 +32767
  - işaretsiz 0 65535
- int (tamsayı) (4 byte)
- long (uzun tamsayı)
  - işaretli -2147483648
  - işaretsiz 0 +65535
- double
- float
- char
- string
- bool

## Declaring (Creating) Variables

### Syntax

~~~C++
    
    // type variableName = value;

    int myNumber = 10;
    double myFloatingNumber = 12.5;
    char character;
    character = 'A';
    string myText = "This is a string variable.";
    bool state = true;

~~~
